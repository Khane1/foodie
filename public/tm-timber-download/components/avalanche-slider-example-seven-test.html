API

To Avalanche Slider plugin offers several callbacks and methods for interacting with the slider.

  • Callbacks
  • Methods
Callback Description
onSetup Called upon slider creation, onSetup: function({})
onLoadEnd Called once all slides are loaded, onLoadEnd: function({})
onSlideBefore Called before slide animation, onSlideBefore: function({})
onSlideAfter Called before slide animation, onSlideAfter: function({})
onSlideshowStart Called once slideshow begins, onSlideshowStart: function({})
onSlideshowStart Called once slideshow begins, onSlideshowStart: function({})
onSlideshowPause Called once slideshow is paused, onSlideshowPause: function({})
onSlideshowEnd Called once slideshow is ended, onSlideshowEnd: function({})
7 Methods

The lighbox offers three public methods, namely .prevContent(), .nextContent(), and .destroyLightbox() which are particularly useful if the lightbox is used as a modal or self navigated projects. Use these medthods for controlling the lightbox from within the loaded content. Works best for inline and ajax loaded content.

Medthod Example Description
prevSlide var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.prevSlide()		 Moves slider to previous slide
nextSlide var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.nextSlide()		 Moves to slider to next slide
slideTo var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.slideTo(index)		 Slides to given slide
initSlideshow var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.initSlideshow()		 Creates and initiates slideshow
pauseSlideshow var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.pauseSlideshow()		 Pauses slideshow
resumeSlideshow var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.resumeSlideshow()		 Resumes slideshow
endSlideshow var slider = $( '.tm-slider-container' ).data( 'avalancheSlider' );
slider.endSlideshow()		 Ends and removes slideshow elements
