Carousel

The slider can easily be turned into a simple carousel of images, showing any number of slides at a given time. All images displayed in a carousel must be of same width and height.

IMPORTANT: Showing more than one slide at the time will automatically disable video APIs and caption scaling/postitioning. In cases where more than one slide is shown and captions are required, use CSS and create specific classes.

Initializing Carousel

To active a carousel style slider set carousel to true and specify the number of slides that should be visible at a time.

Example
								
									$( document ).ready( function(){
										$( '.tm-slider-container' ).avalancheSlider({
											carousel:true,
											carouselVisible:3
										});
									});
