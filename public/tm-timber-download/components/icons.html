Font Icons

Entypo

Timber uses Entypo font icons which are composed of some 250+ icons. Easily style icons using border color classes, background color classes and size classes. Highlight icons using the .feature-icon class.

Sizes

To alter size use one of the four existing size classes: .small, .medium, .large or .xlarge.

Example

Colors

Add border and background color classes in combination with the feature icon class to draw attention to an icon.

Example

Styles

Alternate styles by mixing border and background color class with style classes such as .thick, .dashed, .dotted, .thick, .rounded, and .circled. Hover the icons below to see the result.
